Rosier Photography Project Overview

Project Overview

The purpose of this project is to design and develop a portfolio website for Jackson Derosier's film photography work. The website showcases various collections of film photography with a clean, professional design that matches the artistic aesthetic of the client. It is intended to help the client share his work publicly and build a professional online presence.

Intended Users

  • Potential photography clients
  • Artistic collaborators and galleries
  • Casual viewers and art enthusiasts

Content Overview

  • Home Page: Introduction and dynamic hero imagery
  • About Page: Client background and photography journey
  • Contact Page: Ways to reach out and social media links
  • Collections: Gallery previews for different collections
  • Gallery Pages: Full-size images flex-wrapped neatly

Dynamic Functionality

  • Current Year Display: The footer dynamically displays the current year using JavaScript.
  • Header Color Change on Scroll: The header background color changes subtly when the user scrolls down past 50px, creating a dynamic visual cue.
  • Dynamic Body Height Adjustment: On page load, JavaScript sets the body height based on the viewport height to ensure the footer always sticks to the bottom even with minimal content.

Additional Features

  • Instagram social link integration
  • Flexbox and responsive layout for mobile friendliness
